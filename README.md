# FARE

A premium, minimalist blog dashboard designed for finance professionals. Built with React 19 and Framer Motion for a fluid, high-end user experience.

<img width="3168" height="1999" alt="Screenshot 2026-02-04 211520" src="https://github.com/user-attachments/assets/b39584b7-20c1-48d3-9e40-af511b316855" />


### Core Philosophy
- **Editorial Precision**: Typography-first interface using Playfair Display.
- **Cinematic Motion**: Fluid interactions powered by Framer Motion.
- **Contextual Intelligence**: Real-time reach metrics and creator analytics.
- **Adaptive Interface**: Seamless transition between Paper White and Ink Black.

### Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styles**: Tailwind CSS, Radix UI
- **Motion**: Framer Motion
- **State**: TanStack Query
- **Data**: JSON Server (Mock API)

### Quick Start
```bash
npm install
npm run server # Port 3001
npm run dev
```
