# FARE

A premium, minimalist blog dashboard designed for finance professionals. Built with React 19 and Framer Motion for a fluid, high-end user experience.

<img width="3165" height="1999" alt="Screenshot 2026-02-03 203122" src="https://github.com/user-attachments/assets/1dde2045-ad3e-4caf-a2a4-323d47066ea8" />


## Features

- **Minimalist Design**: A clean, typography-focused UI using Playfair Display and Inter.
- **Fluid Transitions**: Seamless page navigation powered by Framer Motion.
- **Dashboard First**: A centralized hub for metrics, recent journals, and quick actions.
- **Night Mode**: Comprehensive dark theme support for comfortable reading.
- **Content Management**: Intuitive tools for writing and managing articles.
- **Secure Auth**: Integrated Google OAuth.

## Tech Stack

- **Core**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **State**: TanStack Query
- **Backend**: JSON Server (Mock API)

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the Backend (Mock API)**
   ```bash
   npm run server
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```
